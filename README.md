## Variables

**_Variables_** declaration should start with `$` follow by `_` or alphabet.

> **Important:** it is recommended to use camelcase in declaring variables to avoid errors.
```php
<?php

  $firstName = "Mikasa";
  $lastName = "Ackerman";
  echo "$firstName $lastName"; // Mikasa Ackerman
  
?>  
```
